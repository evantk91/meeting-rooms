# meeting-rooms

Given an array of meeting time intervals where intervals[i] = [start_i, end_i],
return the minimum number of conference rooms required.

For example:
```
Input => intervals = [[0, 30], [5, 10], [15, 20]]
Output => 2
```
For example:
```
Input => intervals = [[7, 10], [2, 4]]
Output => 1
```
